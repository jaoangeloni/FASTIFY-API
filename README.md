# API básica com fastify
Projeto simples feito para fins acadêmicos. Lista de todas as tecnologias utilizadas:
  - Node.JS
  - Fastify
  - React
  - TailwindCSS
  - Prisma
  - MongoDB

---
Para testar:
  - Clone o repositório
```
git clone https://github.com/jaoangeloni/FASTIFY-API.git
```
  - Instale as dependências
```
npm install
```
